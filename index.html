Project

We hope these resources make you feel at home!

  • Contributing

    New features or old Bugs?

    We welcome our user community to help us make Varnish better. If you want to report a bug you found or would like to develop a new feature for Varnish the best starting point is reading our Contributor Guidelines https://www.varnish-cache.org/trac/wiki/Contributing

  • Source Code

    Contributions encouraged

    Our repository is in Github https://github.com/varnishcache/varnish-cache/. For incoming patches to the Varnish code base you want to discuss, submit a pull request to Github or send them to the varnish-dev mailing list.

  • Releases

    Current, Stable and Older releases

    An overview of the releases since Varnish 2.0 can be found here https://www.varnish-cache.org/releases

  • VMODs and Utilities

    Extending VCL and adapting Varnish to any environment

    Our fantastic community has written a whole lot of VCL extensions called Varnish Modules https://www.varnish-cache.org/vmods featuring on-the-fly database lookups to do GeoIP and Mobile Device Detection, Manipulate and change Cookies/Headers as well as making calls to external services like cURL, memcached, redis, sqlite. There are also a useful and large set of utilities https://www.varnish-cache.org/utilities that help you adapt your Varnish to any environment and makes its operation much easier featuring Statistics, Monitoring, Server Management, Cache Invalidation and more.
